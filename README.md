
# Weather Mate

Implemented Open Weather API to get current weather and 7 days forecast.
User needs to enter city name to get weather data.



## Features

- Cross Platform
- Live Weather
- Clean and Precise UI
- Live Weather data

**Stack:** React, HTML, CSS, JavaScript, API
